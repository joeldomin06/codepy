¿Que es CodepyMaker?

pagina web que se enfoca en la programacion, esta es planteada en problematicas con diferentes formas de resolución el cual acompañando con ayudas y referencias, mejora y facilita el aprendizaje del lenguaje. Pagina con proyeccion a diferentes lenguajes, principalmente python.



Completar el codigo

Los ejercicios de completar codigo son esenciales para fomentar el aprendizaje debido a que sirve como primeros pasos para comenzar a escribir codigo libremente. Dando ya una idea de base para resolver un problema.


Ordenar el codigo

los ejercicios de ordenar el codigo fomentan el pensamiento logico y mejora la capacidad de secuenciar, el cual es importante para el aprendizaje del lenguaje.


Ruteo

los ejercicios de ruteo son los mas importantes para entrenar la capacidad secuencial, ya que se tiene que seguir el codigo como lo haria el computador el cual te dara una idea de lo que pasara con el programa


Quizes

Los Quizes tienen un rol fundamental en el aprendizaje, ya que se puede demostrar los conocimientos con test especializados en una parte del lenguaje de programación como los for o los condicionales en python


Referencias y ayudas

Con el fin de facilitar el aprendizaje del lenguaje, se colocaron botones de consejos, los cuales te daran tips para poder resolver el ejercicio. La pagina de referencias sirve para señalar las funciones importantes y como funcionan, aparte de que indica recomendaciones de editores de codigo para que sea mas facil la tarea de escritura.
