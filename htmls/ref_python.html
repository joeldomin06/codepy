Variables

Crear variables

Las variables son contenedores para almacenar datos.Python no posee un comando especifico para crear variables, como en otros lenguajes de programación.Una variable es creada cuando por primera vez le asignas un valor.

Ejemplo:

x = 4
y = 'hola'

Además en las variables con strings las comillas simples y las dobles son lo mismo en Python

Nombrar una variable

Una variable puede tener un nombre corto (como x o y) o uno más descriptivo (como edad,fecha,area_total).Sin embargo Python tiene regalas para considerar a una variable:

  • Una variable debe de comenzar con una letra o un '_'

  • Una variable no puede iniciar con un numero

  • Una variable solo puede contener caracteres alpha-numericos y guiones bajos (A-z, 0-9 y _ )

  • Las variables es sensible a las mayúsculas ej: edad, Edad y EDAD son 3 diferentes variables

Salidas de variables

En Python print es el comando para generar una salida de datos.

Ejemplo:

x = 'hola mundo'
print(x)

En la salida:

hola mundo

print() acepta cualquier tipo de dato y lo transforma a un texto para despues mostrarlo

Condicionales

Condiciones:

Estos son los que acompañarán a las declaraciones "if" o "elif" y bucles.

Igual: a == b
No es igual: a != B
Menos de: a < b
Menor o igual que: a <= b
Mayor que: a > b
Mayor o igual que: a >= b

Declaración if:

Se escribe una "declaración if" utilizando la palabra clave if.

Ejemplo:

a = 50
b = 100
if b > a:
  print("b es mayor que a")

En este ejemplo, usamos dos variables, a y b, que se usan como parte de la instrucción if para probar si b es mayor que a. Como a es 50 yb es 100, sabemos que 100 es mayor que 50, por lo que imprimimos en la pantalla que "b es mayor que a".


Elif

La palabra clave elif es la forma de python de decir "si las condiciones anteriores no eran ciertas, entonces intente esta condición".

Ejemplo:

a = 50
b = 50
if b > a:
  print("b es mayor que a")
elif b == a:
  print("b es igual que a")

En este ejemplo, a es igual a b, por lo que la primera condición no es verdadera, pero la condición elif es verdadera, por lo que imprimimos en la pantalla que "a y b son iguales".


Else

La palabra clave else captura cualquier cosa que no esté atrapada por las condiciones anteriores.

Ejemplo:

a = 100
b = 50
if b > a:
  print("b es mayor que a")
elif b == a:
  print("b es igual que a")
else:
  print("a es mayor que b")

En este ejemplo, a es mayor que b, por lo que la primera condición no es verdadera, también la condición elif no es verdadera, así que pasamos a la condición else e imprimimos en la pantalla que "a es mayor que b".


También puedes tener un else sin el elif:

Ejemplo:

a = 100
b = 50
if b > a:
  print("b es mayor que a")
else:
  print("a es mayor que b")

Sangría

Python se basa en la sangría (espacio en blanco al comienzo de una línea) para definir el alcance en el código. Otros lenguajes de programación a menudo usan llaves para este propósito.

Ejemplo:

a = 50
b = 100
if b > a:
print("b es mayor que a")

En este caso al omitirse la sangría se producirá un error.


And

La palabra clave y es un operador lógico, y se utiliza para combinar declaraciones condicionales

Ejemplo:

a = 100
b = 50
c = 150
if a > b and c > a:
  print("c es mayor que b")

En este ejemplo se comprueban las 2 condiciones y si ambas se cumplen imprime "c es mayor que b".


Or

La palabra clave or es un operador lógico y se usa para combinar declaraciones condicionales

Ejemplo:

a = 100
b = 50
c = 150
if c > b or c > a:
  print("C")

En este ejemplo se comprueban las 2 condiciones y si una de las dos se cumple se imprime "C".


Anidado

Puede tener declaraciones if dentro de declaraciones if, esto se llama instrucciones if anidadas.

Ejemplo:

if a > b:
 if c > b:
  print("O")

En este ejemplo se comprueba la primera condición "a > b", si esta se cumple compruba si "c > b", y esi esta última se llega a cumplir imprime "O".



Ciclo while

Con el ciclo while podemos ejecutar un conjunto de declaraciones siempre que una condición sea verdadera, se debe considerar crear un cierre a ste para que no se repita de manera indefinida.

Ejemplo:

i = 1
while i < 6:
 print(i)
 i += 1

El tiempo del bucle depende de la condición, como en este caso la variable "i" va aumentando de 1 en 1 hasta llegar a 6 y así "romper el ciclo" para salir de este.


Sentencia break

Con la declaración break podemos detener el bucle, incluso si la condición es verdadera:

i = 1
while i < 6:
 if i == 3:
  break
 print(i)
 i += 1

Cuando i sea iguala 3 y pase por el condicional if, eso cerrará el ciclo aunque la condición inicial sigue siendo verdadera.


Funciones

Una función es un bloque de código que sólo se ejecuta cuando se le llama.

Puede pasar datos, conocidos como parámetros, en una función.

Una función puede devolver datos como resultado.


Crear una función

En Python una función se define utilizando la palabra clave "def"

Ejemplo:

def una_funcion():
 print("Hola funcion")

Llamado de función

Para llamar a una función, se utilza el nombre de la función a llamar seguido de paréntesis

Ejemplo:

def una_funcion():
 print("Hola funcion")
una_funcion()

Parámetros

La información puede suministrarse a las funciones como parámetro.

Los parámetros se especifican después de que el nombre de la función, dentro de los paréntesis. Puede añadir tantos parámetros como desee, sólo les separan con una coma.

El siguiente ejemplo tiene una función con un parámetro (variable). Cuando se invoca la función, se pasa a lo largo de un nombre de pila, que se utiliza dentro de la función para imprimir el nombre completo

def una_funcion(variable):
 print(variable + " función")

una_funcion("La")
una_funcion("Una")
una_funcion("Nuestra")

Se puede enviar cualquier tipo de datos de parámetro a una función (cadena, número, lista, diccionario, etc.), y será tratado como el mismo tipo de datos dentro de la función, como en este caso que se pasó un string.


Parámetros por defecto

Si llamamos a la función sin parámetros, se utiliza el valor por defecto.

Ejemplo:

def una_funcion(variable = "Una"):
 print(variable + " función")

una_funcion("La")
una_funcion()
una_funcion("Nuestra")

Como se ve en el ejemplo para hacer un parámetro por defecto se debe asignar un valor a la variable de la función, que en caso de ser llamada sin parámtro automáticamente tomará ese.


Valores retornados

Para permitir que una función devuelve un valor, se utiliza la palabra clave "return"

Ejemplo:

def una_funcion(x):
 return x * 5

print(una_funcion(1))
print(una_funcion(2))
print(una_funcion(4))

Asignaciones con palabras clave

Si hay una función donde se desconoce el orden de sus parámetros se pueden asignar sabiendo los nombres de estos.

Ejemplo:

def una_funcion(x,y,z):
 return x * 5 + y * z

print(una_funcion(z = 1 , y = 2 , x = 4))

Ciclo for

Un ciclo for se utiliza para iterar sobre una secuencia (que es ya sea una lista, una tupla, un diccionario, un conjunto, o una cadena).

Con el ciclo for podemos ejecutar un conjunto de instrucciones, una vez por cada elemento de una lista, tupla, set etc.

Ejemplo:

fruits = ["apple", "banana", "cherry"]
for x in fruits:
 print(x)

Esto es aplicable a string de forma similar, solo que estará tomando los valores de los carácteres de izquierda a derecha


Sentencia break

Con la sentencia break podemos detener el ciclo antes de que haya pasado a través de todos los elementos

Ejemplo:

fruits = ["apple", "banana", "cherry"]
for x in fruits:
 print(x)
 if x == "banana":
  break

La función range

Para que el ciclo pase a través de un conjunto de código de un número especificado de veces, se puede utilizar la función range().

La función range() devuelve una secuencia de números, a partir de 0 por defecto, y los incrementos por 1 (por defecto), y termina en un número especificado (el número dado menos 1).

Ejemplo:

for x in range(6):
 print(x)

Esto imprimirá desde el 0 hasta el 5


Otras formas de usar range()

El range de igual forma se le pueden añadir más parámetros numéricos (máximo 3) de esta forma range(i,j,k) donde "i" es el número inicial, "j" es el sucesor del número límite y "k" es el tipo de suma que se realiza desde el valor inicial.

Ejemplo:

for x in range(1,6,2):
 print(x)

Lo que imprimirá serán los números 1 ,3 y 5


Ciclos anidados

Es un ciclo dentro de un ciclo, donde el ciclo interior se realizará la cantidad de veces que iteraciones tenga el exterior

Ejemplo:

adjetivos = ["alto","pequeño"]
personas = ["Alberto es ","Juan es "]
for x in adjetivos:
 for y in personas:
  print(y+x)

En este caso imprimirá que Alberto es alto y pequeño, y lo mismo con juan.


Sentencia pass

Como ocurría anteriormente con funciones y condicionales, si un ciclo for no tiene argumento dentro de este se debe poner pass para así no generar un error

Ejemplo:

for x in range(1,6,2):
 pass

Strings

Se define un string por el o los caracteres rodeados por comillas simples o dobles ("" o '') sienfo "hola" lo mismo que 'hola'.

Ejemplo:

print("Hola")
print('Hola')

En ambos casos se imprime lo mismo (Hola).


Operaciones con string

Los estring se pueden asignar a las variables y operar en ellas, con sumas de strings, multiplicación de un string por un número entero. Recordar que nunca se puede sumar un string con un valor numérico a menos que sea de la forma string + str(valor)

Ejemplo:

palabra = "Hola"
otra_palabra = " y adiós"
print(palabra + otra_palabra)
print(palabra * 2)

En el caso de la suma lo que resultaría sería print("Hola y adiós") y en la multiplicación print("HolaHola")


Índices de un string

Un tring funciona como una matriz, por lo cual tiene índice, siempre partiendo desde el índice 0.

pa = "Hola"
print(pa[0,4,2])

En este caso a la variable pa se le aplican unas llaves de forma [i,j,k], donde "i" será el índice inicial, "j" el sucesor del límite al que puede llegar y "k" la razón e que se van sumando

En el ejemplo se toman [0,4,2] lo que equivale a los índices 0 y 2 y la palabra es "H" = 0 ,"o" = 1 ,"l" = 2 y "a" = 3 por lo que la palabra que resulta será "Hl".


Verificación

Se puede comprobar si un caracter o una palabra está en otro string utilizando la función in, y en contraparte a esta se puede usar el not in de modo que si la respuesta fuera true en el caso anterior en este sería false.

Ejemplo:

texto = "Hola como estás"
p = "como"
if p in texto:
 print("Si")
if p not in texto:
 print("No")
else:
 print("Si")

Como en ambos casos existe p en texto de igual forma imprimirá Si


Plantillas format

Existe una función llamada format() que utiliza plantillas construidas por string y se usan de la siguiente forma

plantilla = "Edad: {} {}"
edad = 36
años = "años"
print(plantilla.format(edad,años))

En este caso lo que ocurrirá será que donde existan las llaves de la forma {} se remplazarán por el string dado como parámentro en ese mismo orden resultando de la forma "Edad: 36 años"


Listas

Es una colección de elementos ordenado que permite ser alterada y que al igual que otros objetos tiene propiedades de matriz.

Las listas pueden contener diferentes elementos como: int,float,string,booleanos,listas,etc.


Crear una lista

Para crear o definir una lista hay 2 formas básicas: la primera es usar simplemente las llaves de "corchetes" [] sobre uno o varios elementos y la segunda es usar la función list() sobre uno varios elementos

Ejemplo

a = [1,"al",5]
b = list(1,"al",5)

Para hacer una copia de una lista existente y que la lista copiada no se altere se hace esto:

a = [1,2]
b = list(a)

Ya que de otro modo lo que se genera son "llaves" que llamarán a la misma base de datos, osea que si se hace b = a, si se llegase a alterar b, a también cambiaría


Concadenación de listas

Las listas se pueden "sumar" entre sí al igual que los string

Ejemplo:

a = list(1,2)
b = list(3)
c = a + b

Entonces el valor de "c" será [1,2,3], ya que lo que hace la suma es la anexación de elementos aunque estos existan en la lista anterior


Índices de una lista

La lista al tener relación con las matrices lleva consigo índices de la forma: lista[i,j,k] , donde "i" será el índice del elemento inicial, "j - 1" el elemento límite al cual llegar y "k" la razón de adición constante.

lista = [0,1,2,3,4,5,6,7]
print(lista[0,7,2])

En este ejemplo lo que debe imprimir es [0,2,4,6] según lo dicho anteriormente


Reasignación a elementos específicos

Se pueden alterar elementos específicos de una lista a través de sus índices de la siguiente manera:

lista = [0,1,2,3,4,5]
lista[5] = 9
print(lista)

Con lo anteriormente dicho se puede llegar a la conclusión de que el único valor que cambió fué el de índice 5, quedando [0,1,2,3,4,9]


Verificación

Al igual que en los string se puede usar la palabra clave "in" para comprobar si un elemento está en una lista

Ejemplo:

lista = [0,1,2,3,4,5]
if 4 in lista:
 print("Sí está")

En este caso la condición será verdadera y por esto imprimirá "Sí está"


Tuplas

Las tuplas son datos que no son modificables y no pueden ser ordenadas. y son escritas con '()'

Crear una tupla

Para crear una tupla nueva se utiliza tuple(), en cambio con valores, se colocan los valores dentro de ()

Ejemplo:

x = tuple() #esto crea una tupla vacia.
y = ('gato','perro','loro')

Acceder a las tuplas

Para acceder al valor de una tupla refiriendo el indice con '[]'

Ejemplo:

y = ('francia','españa','china')
print(y[1])

En la salida:

españa

Al igual que con los strings estos se pueden tener indices negativos.

Rango de indices

Como en los strings o las listas estos tambien se puede obtener un rango de valores

Ejemplos:

z = ('manzana','pera','uva','banana','cerezas','kiwi')
print(z[1:3])

En la salida:

('pera','uva')

Estos tambien pueden ser negativos.

Ciclo atraves de la tupla

Estos pueden ser recorridos por un ciclo for

Ejemplo:

frutas = ('manzana','pera','uva')
for i in frutas:
 print(i)

En la salida:

manzana
pera
uva

Verificar si un elemento existe

Para determinar si existe un elemento en una tupla se utiliza in

Ejemplo:

frutas = ('manzana','pera','uva')
if 'pera' in frutas:
 print('si, pera esta en frutas')
# en la salida imprime: si, pera esta en frutas

Crear una tupla con un elemento

Para crear una tupla con un elemento se debe crear con el elemento seguido de una coma

Ejemplo:

x = ('casa',)

# esto no es una tupla

Unir dos tuplas

Para unir dos tuplas se coloca el operador de adiccion el cual al unir las va añadiendo al final de la primera tupla

tuple1 = ("a", "b" , "c")

print(tuple1+tuple2)

En la salida:

("a","b","c",1,2,3)

Diccionarios

Los diccionarios son una colección de datos, el cual es sin orden, es mutable e indexiable, se escriben con '{}', posee llaves y valores

Ejemplo:

auto = {'marca':'Ford','modelo':'Mustang','año':1946}

Crear un diccionario

Al crear un diccionario se tiene que tener en cuenta que las llaves tienen que ser datos inmutables como strings, tuplas o números, sin embargo los valores que puede contener esta llave puede ser de cualquier tipo de dato. Los diccionarios vacios se crean con dict() o colocando {}

Ejemplo:

d = dict() #esto crea un diccionario vacio
b = {} #esto tambien
c = {['juan','pedro']:'invitados'} #esto arrojara error debido a que las listas son mutables

Acceder a los valores

Para acceder a cierto valor tienes que indicarle la llave con los '[]'

Ejemplo:

x = auto['año']
print(x)

En la salida:

1946

Cambiar valores

Para cambiar valores se debe asignarle al diccionario con la llave indicada el valor a cambiar

Ejemplo:

auto['año'] = 2019
print(auto) #retorna el mismo diccionario solo que la llave año su valor es cambiado de 1946 a 2019

Recorrer un diccionario

Para recorrer un diccionario se utiliza un ciclo for. Este solo pasa por las llaves del diccionario

Ejemplo:

for x in auto
 print(x) #imprime todas las llaves

Pero tambien exite un metodo para obtener los valores y es así:

for x in auto
 print(auto[x]) #imprime todos los valores

Verificar si un elemento esta en un diccionario

Para Verificar se utiliza el in

Ejemplo:

if 'modelo' in auto:
 print('si, esta en auto') #en la salida esta 'si, esta en el auto' debido a que se encuantra en el diccionario como llave

Añadir items

Para añadir items al diccionario se le asigna a el diccionario con la llave que quiere agregar su respectivo valor

Ejemplo:

auto['color'] = rojo
print(auto) #imprime el diccionario solo que con un valor extra al final de este

Eliminar items

Para eliminar items se utiliza la funcion pop()

Ejemplo:

auto.pop(la llave que quiere eliminar)

Archivos

Los archivos son objetos que se pueden leer,actualizar o eliminar con funciones de python

Existen 4 métodos para operar archivos: "r" sirve para leer un archivo, "a" sirve para actualizar un archivo (agregar información), "w" sirve crear un archivo y escribir en el, si ya existe reescribe sobre este y por último el "x" que sirve para crear un archivo.

b = open(nombre_archivo,"a")
c = open(nombre_archivo,"w")
d = open(nombre_archivo,"x")
a.close()
b.close()
c.close()
d.close()

De esta manera se hace, normalmente se usa para archivos txt y el nombre_archivo por ejemplo debe ser así "archivo.txt" para que pueda operarse, y siempre hay querecordar de luego haber leído un archivo o terminado de operar el cerrarlo para evitar errores.
